Elementos del Dex (JSON)
Pega un array JSON con tus elementos. Cada uno admite: name, icon (un emoji, opcional), rarity (common/uncommon/rare/epic/legendary), description, quote (opcional), imageUrl (opcional), weight (1-100, qué tan seguido aparece).
